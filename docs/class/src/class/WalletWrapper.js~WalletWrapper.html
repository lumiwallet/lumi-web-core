    Creating a core for each supported currency type

    Params:

    NameTypeAttributeDescription
    coins Array<{coin: String, type: String}>
    coins[].coin string

    Short name of coin. Supported coins are BTC, ETH, BCH, BTCV, LTC and DOGE
    coins[].type string | number

    Coin type (additional). For BTC supported types are p2pkh and p2wpkh. For ETH type is a account number (by default 0).
