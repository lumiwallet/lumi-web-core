    public async createCoinsCores(coins: Array<{coin: String, type: String}>): * source

    Creating a core for each supported currency type

    Params:

    NameTypeAttributeDescription
    coins Array<{coin: String, type: String}>
    coins[].coin string

    Short name of coin. Supported coins are BTC, ETH, BCH, BTCV and DOGE
    coins[].type string | number

    Coin type (additional). For BTC supported types are p2pkh and p2wpkh. For ETH type is a account number (by default 0).

    public getChildNodes(data: Object): Array source

    The method returns information about child nodes by the derivation path and range

    Params:

    NameTypeAttributeDescription
    data Object
    data.from number

    Top of the derivation range
    data.to number

    End of the derivation range
    data.path string

    Derivation path
    data.coins Array

    Array of coins for generating addresses. Includes the coin param (e.g. BTC) and the type param (e.g. p2pkh or account number) parameters

    Return:

    Array

    list - Array of child nodes. Every child node contains the following parameters: derivation path, publick key, private key in WIF format and addresses if a list of coins was sent
