    public async calculateFee(params: Object): Promise<Array> source

    This method generates a list of transaction fees for the selected currency

    Params:

    NameTypeAttributeDescription
    params Object
    params.currency string

    Selected currency. BTC, ETH, BCH or BTCV
    params.addressType string

    Address type. Required for BTC transaction. Supported types are p2pkh or p2wpkh
    params.account string

    Account number. Required for ETH transaction
    params.amount number

    Amount of transaction
    params.customFee number

    Custom fee per byte
    params.size number

    Transaction size. Relevant for bitcoin transactions

    Return:

    Promise<Array>

    The list of transaction fees

    public async createCoins(coins: Array<{coin: String, type: String}>): Object source

    Creating a core for each supported currency type

    Params:

    NameTypeAttributeDescription
    coins Array<{coin: String, type: String}>
    coins[].coin string

    Short name of coin. Supported coins are BTC, ETH, BCH and BTCV
    coins[].type string | number

    Coin type (additional). For BTC supported types are p2pkh and p2wpkh. For ETH type is a account number (by default 0).

    Return:

    Object

    core.ETH - A ETH core that contains node, private and public keys and user address
